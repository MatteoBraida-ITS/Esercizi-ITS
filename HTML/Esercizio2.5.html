HTML:
HTML è l'acronimo di HyperText Markup Language, ovvero Linguaggio di markup per ipertesti. È il linguaggio standard utilizzato per creare e strutturare documenti progettati per essere visualizzati in un browser web. Definisce il contenuto e la struttura di una pagina web, utilizzando elementi chiamati tag, racchiusi tra parentesi angolari
CSS:
Cascading Style Sheets, ovvero "fogli di stile a cascata", ed è un linguaggio fondamentale nel web design che controlla l'aspetto grafico (colori, layout, font, animazioni) delle pagine web, separando la presentazione dal contenuto (HTML) per rendere i siti visivamente accattivanti e adattabili a diversi dispositivi
Browser:
Programma per navigare in Internet; il programma serve a inoltrare la richiesta di un documento alla rete e ne consente la visualizzazione una volta arrivato.
Server:
Un server è un computer o software potente che fornisce risorse, servizi e dati ad altri dispositivi (client) su una rete, gestendo richieste simultanee per file, applicazioni o siti web; è ottimizzato per alte prestazioni, affidabilità e sicurezza, fungendo da centrale di controllo e archiviazione. Può essere un hardware specifico, un normale PC adibito a tale ruolo, o un programma software, ed è la base dell'architettura client-server.
